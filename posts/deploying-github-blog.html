Deploying a github blog with nextjs

January 15, 2025

  1. Create a github repository with the same name as your account name.

  2. Clone the repository and initialise with a nextjs blog starter

bash
1npx create-next-app@latest nextjs-blog \
2--use-npm \
3--example "https://github.com/vercel/next-learn/tree/main/basics/dynamic-routes-starter"
  1. Install and confgiure gh-pages
bash
npm install gh-pages
  1. Run deploy!
bash
npm run build && gh-pages -d out --nojekyll
