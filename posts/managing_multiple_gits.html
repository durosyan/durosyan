Managing Multiple Git Repositories

January 15, 2025

I had several folders without initialized Git repositories that I needed to upload to GitHub. To automate this process, I used the following script to set up and push all the repositories to their remote origins.

Note: I used a separate script to create all the remote repositories on GitHub.

This process was quick and efficient with the help of AI.

sh
1#!/bin/sh
2
3# Replace these variables with your own values
4OWNER="..."
5TOKEN="..."
6
7# Define color codes
8GREEN='\033[0;32m'
9RED='\033[0;31m'
10YELLOW='\033[1;33m'
11BLUE='\033[0;34m'
12NC='\033[0m' # No Color
13
14# Get all repositories for the user
15repos=$(curl -s -H "Authorization: token $TOKEN" \
16-H "Accept: application/vnd.github.v3+json" \
17"https://api.github.com/user/repos?per_page=100" | jq -r '.[].name')
18
19# Loop through each repository and get the SSH remote URL
20for REPO in $repos; do
21    # GitHub API URL
22    URL="https://api.github.com/repos/$OWNER/$REPO"
23
24    # Get the repository details
25    ssh_url=$(curl -s -H "Authorization: token $TOKEN" \
26    -H "Accept: application/vnd.github.v3+json" \
27    $URL | jq -r '.ssh_url')
28
29    if [ -n "$ssh_url" ]; then
30        echo "${BLUE}Repository '$REPO' SSH URL: $ssh_url${NC}"
31    else
32        echo "${RED}Failed to get SSH URL for repository '$REPO'${NC}"
33    fi
34
35    # Find the repository in the folder structure based on the repo name
36    repo_path=$(find /Volumes/Extreme\ SSD/Projects -type d -name "$REPO" 2>/dev/null)
37
38    if [ -n "$repo_path" ]; then
39        echo "${GREEN}Found repository '$REPO' at path: $repo_path${NC}"
40    else
41        echo "${YELLOW}Repository '$REPO' not found in the folder structure${NC}"
42    fi
43
44    if [ -n "$repo_path" ]; then
45        cd "$repo_path" || continue
46        git remote set-url origin "$ssh_url"
47        echo "${GREEN}Updated remote URL for repository '$REPO' to '$ssh_url'${NC}"
48
49        # Add all files and commit them
50        git add -A
51        git commit -m "init"
52        git branch -M main
53        git push -u origin main
54    fi
55done
